Personal Projects
Bleeding Out 2D Concept

Bleeding Out 2D Concept is a 2D boss rush game with the original concept being inspired by Shadow of the Colossus. The goal of this project was to create an engaging combat system for the player. This was achieved with a fast paced combo system, animation canceling, and a dodge with I-frames. Another goal was to create an unpredictable but fair enemy AI. This was done by creating a large pool of attacks to pick from, and choosing the best attacks based off of the distance from the player, and the previous attacks in that sequence.

Credit to ShadowIceman
Project R

Project R is a 3D racing game, heavily inspired by Kirby Air Ride, where you have vehicles that both drive on the ground and glide through the air. There are currently 8 different vehicles, each with their own acceleration curve, top speed, turn speed, weight, drift strength, and boost strength. The gliding works by taking each machines weight and top speed stat, as well as their current speed, to create a variable gravity system which makes the vehicles descend at different speeds.
